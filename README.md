## 📝 How to Deploy

### Step 1: Enable GitHub Pages

1. Go to repository **Settings**
2. Navigate to **Pages** (left sidebar)
3. Under "Build and deployment" section:
   - Select **Deploy from a branch**
   - Choose branch: `main`
   - Choose folder: `/ (root)`
4. Click **Save**

### Step 2: Access Your Live Site

GitHub will automatically build and deploy your site. After a few moments (typically 30-60 seconds):

**Your site will be live at:** `https://wuodolo80-max.github.io/PROTECT-Application/`

### Step 3: Custom Domain (Optional)

To use a custom domain:
1. In Settings → Pages, enter your domain name
2. Update your domain registrar's DNS records to point to GitHub
3. GitHub handles SSL certification automatically
